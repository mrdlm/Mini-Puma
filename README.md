# Mini-Puma

A minimal HTTP server implementation inspired by [Puma](https://github.com/puma/puma).

## About

This project is a learning exercise to understand how web servers work by building a simplified version of Puma from the ground up.

## Goals

- Understand HTTP protocol parsing
- Learn about socket programming in Ruby
- Explore concurrent request handling
- Study the architecture of production web servers
